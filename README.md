# Earth

综合地球物理联合反演与解释一体化平台

## 项目初始化（需要 node 环境）

node 安装与配置请参考[https://www.cnblogs.com/xinaixia/p/8279015.html](https://www.cnblogs.com/xinaixia/p/8279015.html)

### \$ `npm install`

这是在安装项目所依赖的第三方包

## 项目运行

### \$ `npm start`

在开发环境运行此 app.<br>
在浏览器中打开 [http://localhost:3000](http://localhost:3000) 来查看.

## 项目打包

### \$ `npm run build`

将项目打包到 `build` 文件夹.<br>
